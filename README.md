# Automation-Testing-of-Contact-Us-Page-AutomationExercise.com
This project automates the Contact Us page functionality on AutomationExercise.com  using Selenium WebDriver with Java in IntelliJ IDEA.
The purpose of this project is to showcase automation testing skills including element handling, form validation, test data management, and result verification using Selenium and Java (or Python, depending on what you used).
Verify navigation to the Contact Us page

✅ Validate visibility and accessibility of all input fields (Name, Email, Subject, Message, File Upload)

✅ Test form submission with valid data

✅ Test form submission with invalid or empty fields

✅ Verify the success confirmation message after form submission

✅ Validate file upload functionality

✅ Check error handling and message validation

✅ Test “Home” button redirection from the contact page

Tools & Technologies

Language: Java 

Framework: Selenium WebDriver

Test Runner: TestNG / JUnit

Browser: Chrome 

Build Tool: Maven 

IDE: IntelliJ IDEA
